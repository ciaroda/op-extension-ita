Procedura di installazione di un'estensione Chrome non pacchettizzata (Unpacked Extension)
Prerequisiti
Google Chrome installato.
Cartella dell'estensione già disponibile sul computer (contenente il file manifest.json).
Procedura
Aprire la pagina delle estensioni

Aprire Google Chrome e digitare nella barra degli indirizzi:

chrome://extensions

Premere Invio.

Abilitare la Modalità sviluppatore

Nell'angolo in alto a destra attivare l'interruttore:

Modalità sviluppatore (Developer mode)

Caricare l'estensione

Compariranno tre pulsanti nella parte superiore della pagina.

Fare clic su:

Carica estensione non pacchettizzata (Load unpacked)

Selezionare la cartella dell'estensione

Nella finestra di selezione:

individuare la cartella contenente l'estensione;
selezionare la cartella principale (non il singolo file manifest.json);
fare clic su Seleziona cartella.
Verificare l'installazione

Se l'estensione è valida:

comparirà nell'elenco delle estensioni;
sarà automaticamente abilitata;
la relativa icona potrebbe essere visibile nella barra degli strumenti di Chrome (oppure tramite il menu delle estensioni).
